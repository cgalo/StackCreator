Author: CGalo
IDE: IntelliJ
JDK: Java 11

Description:
Java program that allows the user to create a stack. Menu offers functions for: 
- Adding to stack
- Deleting peek 
- What is current stack's peek
- What is stack's bottom
- Display what's in the stack
- Empty stack
- Quit program

Objective: 
Understand and implement stack on a small project to implement stacks on further projects
